# Hikoo Backend

## Prerequisite
- node.js version 10+
- npm


## Installation

```bash
$ npm install --no-save
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## API
```bash
$ npm run start:dev

$ google-chrome http://localhost:3000/api
```
